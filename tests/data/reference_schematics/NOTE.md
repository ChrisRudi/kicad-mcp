# Referenz-Schaltbilder (Goldstandard)

Professionell gezeichnete KiCad-Schaltbilder, gegen die die Layout-Metrik
(`generators/schematic/layout_measure`) geeicht ist: die Profis müssen
**badness = 0** erreichen. Sie dienen (a) als Kalibrier-Anker im Test, (b) als
Referenz für die „Distanz zum Profi"-Fitness.

Quelle: offizielle **KiCad-Demos** (`sallen_key`, `rectifier`) aus dem
KiCad-Quellbaum (gitlab.com/kicad/code/kicad, `demos/simulation/`), GPL —
kompatibel mit diesem GPL-3.0-Projekt. Unverändert übernommen.
